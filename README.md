# Learning Recommendation System

An ML-powered platform that recommends learning resources based on student performance and topics.

## Project Overview

This project is a full-stack application designed to help students discover personalized learning resources. Students can:
- Register and create a profile
- Log learning topics and performance
- Receive AI-powered recommendations for resources
- Track their learning progress
